== Authentication
|===
| header | description

| `Authorization`
| API를 사용하기 위한 인증 토큰

| `Refresh`
| access token을 재발급 받기 위한 토큰
|===

== Business Error Code
|===
| code | status |description

| `EXIST_EMAIL`
| 400
| 이미 가입한 e-mail입니다.

| `NOT_FOUND_USER`
| 400
| 찾을 수 없는 회원입니다.

| `NOT_FOUND_POST`
| 400
| 찾을 수 없는 게시글입니다.

| `NOT_MATCHING_OWNER`
| 400
| 작성자와 동일한 회원이 아닙니다.
|===

== 공통 Response
|===
| field | 설명

| `time`
| 이벤트 발생 시간

| `data`
| 반환될 데이터
|===

== 공통 ErrorResponse
|===
| field | 설명

| `time`
| 이벤트 발생 시간

| `status`
| 상태 코드

| `msg`
| 에러 메시지
|===
