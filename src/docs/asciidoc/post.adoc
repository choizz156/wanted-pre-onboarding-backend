==== `(2) 여러 개 조회`
- page 파라미터가 1보다 작을 경우 예외가 발생합니다.
```json
{
    "time": "2023-08-09T07:08:12.046862",
    "data": {
        "status": 400,
        "msg": "paging.page: must be greater than or equal to 1"
    }
}
```

===== 파라미터가 null일 경우 파라미터 기본값
|===
| parameter | default

| `size`
| 10

| `sort`
| DESC
|===
