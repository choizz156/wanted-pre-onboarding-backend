==== `(1) 유효성 검사`
|===
| path | description

| email
| `이메일은 @가 포함되어야 합니다. ex)test@gmail.com`

| `패스워드는 8자리 이상이어야 합니다.`

    HTTP/1.1 400 Bad Request

   {
    "time": "2023-08-06T03:48:07.811118",
    "status": 400,
    "customFieldErrors": [
        {
            "field": "email",
            "rejectedValue": "testgmail.com",
            "reason": "@가 포함되어야 합니다."
        }
    ]
}
